
## Description

This AutoHotkey script is a key-bindings like a macOS.

## Prerequisite

* [AutoHotkey](https://autohotkey.com/)

## Supported bindings

* Moving
    * [x] Ctrl-a
    * [x] Ctrl-e
    * [x] Ctrl-f
    * [x] Ctrl-b
    * [ ] ...
* History
    * [x] Ctrl-p
    * [x] Ctrl-n
    * [ ] ...
* Changing Text
    * [x] Ctrl-d
    * [ ] ...
* Killing and Yanking
    * [x] Ctrl-k
    * [x] Ctrl-y
    * [ ] ...

## Usage

Double click a `*.ahk` file.
